Stone Pagamentos

Rio de Janeiro

22 de Julho de 2017

Você sabe muito bem que a tecnologia é a melhor solução pra qualquer problema. Aqui, colocamos isso em prática, e o primeiro passo foi o mercado de pagamentos. A verdade é que estamos apenas começando e a ideia é mudar o mundo todo. Afinal, “software is eating the world”, e isso é incrível.

A Stone tem um propósito claro: usar inteligência e as melhores tecnologias pra resolver os problemas financeiros dos nossos clientes.

''Talk is Cheap, show me the code!’’

Não é que não existam regras, é que elas são construídas de acordo com o problema a ser resolvido. Assim como nossos DEVs possuem perfis diferentes entre si, apostamos na variedade de linguagens pra desenvolver nossas soluções. Nossa stack envolve: -> Vue.Js -> C# -> React -> .NET -> Go -> JavaScript -> AngularJS -> Node.js -> MongoDB -> Python -> Elastic Search E muito mais coisa! Mas relaxa, você não precisa saber tudo isso!

E aí, pronto pra fazer parte do nosso time? Entra em http://enter.stone.com.br e manda seu currículo!
